# Asset notice — editorial

## texas-outline.svg

Original vector outline of Texas, traced by hand from public-domain state
boundary coordinates (U.S. Census Bureau TIGER/Line–style latitude/longitude
vertices) and projected with a simple equirectangular projection into the
SVG `viewBox="0 0 116 96"` coordinate system.

- Source data: public-domain geographic boundary facts (state borders are not
  copyrightable). No third-party SVG file was copied or hotlinked.
- Projection and vertex simplification are our own.
- License: public domain. No attribution required.

The same coordinate system is reused inline by
`src/components/TexasHeroMap.tsx`, so the hero map's city labels are positioned
in real geographic space — not with arbitrary CSS percentage overlays.
